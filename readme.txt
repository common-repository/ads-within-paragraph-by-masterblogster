== Description ==

Ads within paragraph plugin helps you add your advertisement inside wordpress article, you can add your advertisement left to, right to, above or below any paragraph. You can add your AdSense, Yahoo, Bing, Media.net or any other advertisement script with Ads within paragraph plugin.

We are using ads within paragraph plugin in our blog- <a href="http://masterblogster.com">MasterBlogster</a>.

For more information and screenshots check out the <a href="http://masterblogster.com/plugins/ads-within-paragraph/">plugin homepage</a>.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
